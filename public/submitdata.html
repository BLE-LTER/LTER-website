Data Submission Guidelines


Got some great data? Hooray! Now let's share it with the world. Remember that BLE data must be archived within two years of collection. Be sure to deliver data to the information manager at least two months before you need it archived. Here's how.


1

Let us know you'd like to submit data.

We'll guide you on how to organize your data for the archive, and share tips for an optimal data publishing experience.

2a

New data: send us your data and a completed metadata template.

Be sure to download the latest template from the link above.

2b

Update to existing data archive: send us your data and see this FAQ.

3

Stay in touch.

Until the data package is officially archived, be available to answer questions we may have.


Once the dataset is archived and has a DOI, please cite the data package in any related publications, both in a data dissemination section and in the references section. You might want to use the suggested citation available on the data archive's landing page, but please make sure Beaufort Lagoon Ecosystems LTER is in the citation. Thank you!

Sincerely,

The BLE LTER IM team
